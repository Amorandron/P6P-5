## Getting Started
### Parser
To build parser execute
```sh
$ ./gradlew jar
```
Execute parser with:
```sh
$ <path to parser.jar> <list files directory> -a
```

To also insert into the database:
```sh
$ <path to parser.jar> <list files directory> -ai
```
### Chatbot
To build parser execute
```sh
$ ./gradlew jar
```
Run chatbot with:
```sh
$ <path to parser.jar> <discord token> 
```
